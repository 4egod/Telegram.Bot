File Class
This object represents a file ready to be downloaded. The file can be downloaded via GetFileAsync(String, Stream, CancellationToken). It is guaranteed that the link will be valid for at least 1 hour. When the link expires, a new one can be requested by calling GetFileAsync(String, Stream, CancellationToken).
