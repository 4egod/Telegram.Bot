This object represents an incoming callback query from a InlineKeyboardButton. If the button that originated the query was attached to a Message sent by the bot, the field message will be presented. If the button was attached to a message sent via the bot (in inline mode), the field InlineMessageId will be presented.

This object represents a file ready to be downloaded. The file can be downloaded via GetFileAsync(String, Stream, CancellationToken). It is guaranteed that the link will be valid for at least 1 hour. When the link expires, a new one can be requested by calling GetFileAsync(String, Stream, CancellationToken).
