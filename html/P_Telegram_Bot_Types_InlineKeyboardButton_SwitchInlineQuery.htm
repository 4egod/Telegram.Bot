Remarks
Note: This offers an easy way for users to start using your bot in inline mode when they are currently in a private chat with it. Especially useful when combined with switchPm[...] parameters (see AnswerInlineQueryAsync(String, InlineQueryResult, NullableInt32, Boolean, String, String, String)) – in this case the user will be automatically returned to the chat they switched from, skipping the chat selection screen.
